qDNA.utils.calc_ipr_dm(dm: ndarray) → float[source]

Calculates the inverse participation ratio (IPR) of a density matrix.

IPR = :math:´( sum_{i,j}

ho_{i,j} )^2 /( N sum_{i,j} ho_{i,j}^2 )´.

Localized state: IPR = 1/N Delocalized coherent state: IPR = N Maximally mixed state (1/N on diagonal, no off-diagonal elements): IPR = 1

Args:

density_matrix (np.ndarray): Density matrix.

Returns:

float: The inverse participation ratio of the density matrix.
